Тема теоретической части: Использование Python и Tkinter для разработки настольных приложений с базами данных SQLite.
Задание на практическую часть: Разработка программного комплекса "National Cuisine Calculator" для управления рецептами мировых кухонь, расчёта ингредиентов и ведения статистики.

ОПИСАНИЕ ФАЙЛОВ ПРОЕКТА:
1. main.py - Точка входа в приложение, инициализация главного цикла программы.
2. database/ - Пакет для работы с базой данных:
   - database.py: Менеджер БД, создание таблиц, архивация, основные запросы.
   - seed_recipes.py, seed_recipes_extended.py: Начальные данные для заполнения каталога.
3. core/ - Ядро системы:
   - app.py: Главный контроллер, навигация, переключение языков, заставка.
   - translations.py: Словарь переводов (Русский/Арабский) и справочная система.
4. recipes/ - Модуль управления рецептами:
   - recipe_management.py: Каталог, поиск, просмотр карточек, добавление/удаление.
   - personal_recipes.py: Создание личных рецептов пользователями.
5. calculator/ - Модуль расчётов:
   - calculator_view.py: Алгоритм пересчёта ингредиентов и экспорт результатов в TXT.
6. cuisines/ - Справочник кухонь:
   - cuisines_view.py: Описание 15 национальных кухонь мира.
7. stats/ - Модуль статистики:
   - statistics_view.py: Формирование отчётов по категориям и кухням.
8. users/ - Управление пользователями:
   - user_management.py: CRUD-интерфейс для администратора.
9. database.db - Файл базы данных SQLite (создается автоматически).
10. README.md - Подробная документация проекта (Report).
11. USER_GUIDE_RU.md / USER_GUIDE_AR.md - Руководства пользователя.
